# Strategy AI — что осталось для полноценного выпуска в продакшн

## ✅ Сделано в этой сессии

- **beforeunload** — предупреждение при закрытии вкладки с несохранёнными изменениями
- **ConfirmDialog вместо confirm/alert** — удаление карты, проекта, восстановление версии
- **Офлайн-баннер** — уведомление при потере соединения
- **Кнопка «Повторить»** — при ошибке автосохранения в MapEditor
- **Синхронизация Splash с auth** — сплэш ждёт завершения проверки авторизации
- **Последний проект/карта** — блок «Продолжить с…» на странице проектов
- **Кнопка «Назад» в браузере** — history.pushState и popstate для навигации

---

## P0 — обязательно до запуска

| Задача | Описание |
|--------|----------|
| **API-ключ Anthropic/OpenAI** | Задать `OPENAI_KEY` (или `ANTHROPIC_KEY`) на сервере. Без него AI не работает. |
| **Реальная оплата** | Подключить Stripe/Paddle по `PAYMENT_INTEGRATION.md`: Checkout, webhook, обновление тарифа. |
| **JWT_SECRET** | Задать уникальный `JWT_SECRET` в production (не дефолтный). |
| **DATABASE_URL** | PostgreSQL должен быть настроен и доступен. |
| **ALLOWED_ORIGINS** | Указать домен приложения в CORS (например `https://strategy-ai.app`). |

---

## P1 — сразу после запуска

| Задача | Описание |
|--------|----------|
| **Мониторинг ошибок** | Настроить Sentry (`SENTRY_DSN`) для отлова ошибок в production. |
| **Email-сервис** | Настроить отправку писем (верификация, сброс пароля, напоминания). |
| **Верификация email** | Реальная отправка ссылки подтверждения при регистрации. |
| **Обработка ошибок API** | Добавить toast при ошибках удаления проекта/карты (сейчас тихий catch). |

## P2 — когда будет время

| Задача | Описание |
|--------|----------|
| **Еженедельный AI-брифинг** | Модалка с итогами недели и AI-саммари. |
| **Email/Push уведомления** | Использовать тоглы в профиле для подписки на события. |
| **Лимиты при даунгрейде** | Предупреждение и выбор карт при понижении тарифа. |
| **Единые отступы** | Константы `--radius-sm`, `--radius-md` в CSS. |
| **Светлая тема** | Проверить все экраны на захардкоженные тёмные цвета. |

## Инфраструктура
